# 功能目标

首页作为默认 tab，聚合展示当日收益、摸鱼/工作计时、近期关键日期与人生航程摘要，是用户每天进入产品后的核心总览页。

# 共享领域概念

| 名词 | 说明 |
| --- | --- |
| 今日会话 | 当日工作时长、避风时长和当前计时模式。 |
| 收益快照 | 基于档案配置计算出的时薪、秒薪、今日收益与月度累计。 |
| 潮汐任务预览 | 动力室中的今日任务完成摘要。 |
| 岁月坐标预览 | 来自时间轴的近期重要日期。 |
| 人生航程看板 | 基于档案日期计算出的已航行、距离退休、总航程与余辉摘要。 |

# 共享组件

- 收益总览卡
- 计时控制卡
- 摘要统计卡
- 重要日期列表卡
- 人生航程卡组
