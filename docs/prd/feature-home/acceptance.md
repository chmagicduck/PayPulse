# 验收清单

## 功能

- [ ] 首页可展示收益总览、倒计时、计时控制、任务预览、常规波动、岁月坐标和人生航程。
- [ ] 切换工作/避风计时后，累计时长和收益会同步更新。
- [ ] 手动修正避风时长后，统计结果立即刷新并落盘。
- [ ] 收益隐藏开关在重新进入页面后保持上次选择。

## UI

- [ ] 首页保留卡片式信息密度与层级，不出现长表单化布局。
- [ ] 信息块优先复用共享卡片和按钮模式，不把原型展示动画写成必需交互。
- [ ] 关键数字使用统一的强调 token 与等宽数字样式。

## 可维护性

- [ ] 计算公式全部可从 `profile.settings` 与 `home.daily.session` 推导。
- [ ] 岁月坐标与潮汐任务仅做预览，不在 home 内重复维护完整数据。
- [ ] 页面文件只负责装配和事件绑定，计时存取逻辑可下沉到 feature model。
