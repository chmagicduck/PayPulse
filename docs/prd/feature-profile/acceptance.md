# 验收清单

## 功能

- [ ] 个人基地页能展示头像、昵称、用户 ID、等级摘要和三组入口。
- [ ] 头像选择后能立即回显，并在重新进入页面后保持一致。
- [ ] 档案设置页保存后，首页与其他依赖页面读取到同一份配置。
- [ ] 工作制、上下班时间、午休开关与午休时间的约束被校验。

## UI

- [ ] 个人基地优先复用 `page-shell`、`section-title`、`pp-card`、`pp-button` 等共享结构。
- [ ] 个人身份区、控制台区、信息列表区层次清晰，不混用业务色块与局部硬编码。
- [ ] 弹层与底部保存区遵循统一圆角、阴影与间距 token。

## 可维护性

- [ ] 所有可编辑字段都已登记到 `profile.settings`。
- [ ] 等级展示明确标注为读取 `lab.progress` 的摘要，不在 profile 内重复维护。
- [ ] 页面事件只负责表单装配、导航与局部视觉状态，不承载业务计算。
