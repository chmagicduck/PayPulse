# 功能目标

社区页用于承接官方社群加入与权益说明，帮助用户理解社群价值并进入官方交流渠道。

# 页面清单

| --- | --- | --- |
| join-community | features/community/pages/join/index | docs/prototype/feature-community/page-join-community.ts |

# 共享领域概念

| 名词 | 说明 |
| --- | --- |
| 官方社群 | 产品官方维护的交流入口。 |
| 社区权益 | 经验交流、功能抢先看、需求反馈等权益说明。 |
| 加群引导 | 用户通过企业微信插件与官方群链接加入官方社群的行动入口。 |

# 共享组件

- 社区头图区
- 社群加入卡
- 权益说明卡
