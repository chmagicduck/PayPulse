# 验收清单

## 功能

- [ ] 页面可展示官方社群定位、基础数据和权益说明。
- [ ] “立即加入”入口通过企业微信插件和官方群链接承载，或在异常时给出明确降级说明。
- [ ] 从个人基地可稳定进入社区页。
- [ ] 页面支持分享给好友与分享到朋友圈。

## UI

- [ ] 头图区、加入卡和权益卡有明显层级差异。
- [ ] 加入按钮是页面主 CTA，不被次要信息抢焦点。
- [ ] 统计卡和权益卡在窄屏下仍可读。

## 可维护性

- [ ] 加群入口描述与企业微信插件接入配置保持一致。
- [ ] 社区统计与权益文案集中管理。
- [ ] 页面不依赖其他 feature 的内部实现。
