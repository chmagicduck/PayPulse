# 验收清单

## 功能

- [ ] 可在收益与时长两个维度间切换趋势图。
- [ ] 可在日、周、月、年四种范围间切换占比统计。
- [ ] 可查看近 7 日明细并对单条时长进行修正。
- [ ] 修正后的时长会同步影响收益换算与统计展示。

## UI

- [ ] 图表区、成就区、占比区和明细区层次清晰。
- [ ] 图表在无数据和少量数据时仍可读，不依赖原型中的随机 mock 才成立。
- [ ] 修正弹层的输入结构与首页修正时长保持一致。

## 可维护性

- [ ] 图表切换只改变展示维度，不产生新的业务状态。
- [ ] 历史修正数据统一登记在 `report.history.adjustments`。
- [ ] 趋势聚合、占比统计与收益换算可被独立测试。
