# 功能目标

动力室负责承载任务、快乐值、等级进度与勋章体系，为产品提供激励反馈与成长感。

# 共享领域概念

| 名词 | 说明 |
| --- | --- |
| 快乐值 | 完成任务后累计的核心积分。 |
| 等级体系 | 快乐值映射出的等级与下一等级距离。 |
| 日常巡航任务 | 每日可重复或有限次完成的行为项。 |
| 荣誉勋章 | 长周期成就目标。 |

# 共享组件

- 等级进度卡
- 任务列表项
- 勋章进度项
- 等级节点预览条
