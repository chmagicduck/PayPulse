# 验收清单

## 功能

- [ ] 展示当前等级、距离下一等级的进度和今日获得的快乐值。
- [ ] 支持完成日常任务并累加快乐值。
- [ ] 支持查看勋章目标与进度。
- [ ] 个人基地可读取到正确的等级摘要。

## UI

- [ ] 等级卡、任务列表和勋章列表层次清晰。
- [ ] 已完成任务与可继续完成任务的状态区分明确。
- [ ] 原型中的自动预览仅保留为可选浏览体验，不作为强制行为。

## 可维护性

- [ ] 等级计算与任务进度都集中登记在 `lab.progress`。
- [ ] 跨日重置规则有明确入口，不散落在页面组件内。
- [ ] 任务和勋章配置可独立维护。
