# 验收清单

## 功能

- [ ] 能展示当前本地存储占用和容量占比。
- [ ] 能触发数据导出并给出处理中/成功/失败反馈，成功后 JSON 已复制到剪切板。
- [ ] 能触发数据重置，并要求二次确认后才执行。
- [ ] 重置后首页、档案、时间轴、动力室等数据被统一清空。

## UI

- [ ] 危险操作与普通操作在视觉上明确区分。
- [ ] 提示文案能说明“本地优先存储”和“不可撤销”风险。
- [ ] 云端同步区以占位态呈现，不伪装成已可用功能。

## 可维护性

- [ ] 导出与重置逻辑不直接散落到多个页面。
- [ ] 页面只承载状态展示与确认流，不定义各 feature 的业务 schema。
- [ ] 占位能力与正式能力边界清楚。
