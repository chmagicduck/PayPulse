# 业务模型

## 全局业务原则

- 本版采用本地优先策略，正式业务数据默认写入本地存储。
- 共享规则只定义一次，不允许首页、战报、日历分别维护不同计算口径。
- 页面是消费和触发层，真正的共享业务规则要能脱离页面单独描述。
- 占位能力必须显式标为“展示占位”，不得在交互文案上伪装成已可用能力。

## 航行档案是全局输入源

`profile.settings` 是 1.0.0 的基础输入源，直接影响：

- 首页收益计算与倒计时
- 日历发薪日与工作制推导
- 洋流战报的收益换算
- 岁月坐标的人生类摘要

航行档案的业务规则：

- `monthlySalary` 必须大于 `0`
- `payDay` 必须在 `1-31`
- `endTime` 必须晚于 `startTime`
- 启用午休时，午休区间必须落在工作时段内且结束时间晚于开始时间
- `workMode` 非 `big-small` 时，不保留大小周临时切换结果

## 今日会话与收益规则

首页维护今日会话，战报消费聚合结果。

规则如下：

- `workSeconds` 与 `moyuSeconds` 都不得为负数
- 首页允许在工作态和避风态之间切换，并按秒推进当前会话
- 手动修正时长只能写入合法的时分秒
- 跨日后需要重置当日会话，避免把前一天累计到新一天
- 收益口径沿用旧 PRD 约定：`todayEarned = (workSeconds + moyuSeconds) * secondRate`
- 倒计时依赖航行档案的上下班时间，缺省时使用默认作息

## 日历推导规则

日历页不拥有独立业务存储，按共享输入派生展示。

输入包括：

- `profile.settings.payDay`
- `profile.settings.workMode`
- `profile.settings.isCurrentBigWeek`
- `time.axis.entries`
- 2026 年法定节假日与补班预置常量

规则如下：

- 当 `payDay` 超出当月天数时，发薪日取当月最后一天
- 工作制决定普通周末与大小周的工作/休息口径
- 2026 年法定节假日和补班数据由本地预置常量统一维护
- 选中日期必须始终落在当前月可见范围内

## 岁月坐标规则

岁月坐标是全局日期类事实源，首页和日历都只消费它，不复制记录。

规则如下：

- 坐标条目至少包含标题、日期、分组、图标和提醒模式
- `isAnniversary=true` 时按周年逻辑计算下一次提醒
- `isAnniversary=false` 时允许显示“已过 N 天”
- 本版只支持公历录入，不提供农历选项和换算入口
- 首页只展示近期若干条预览，不在首页直接维护条目

## 动力室成长规则

动力室负责成长体系，个人基地只展示摘要。

规则如下：

- 每个任务奖励必须为正数
- 任务完成次数不得超过上限
- 每次完成任务时，`totalHappiness` 与 `todayHappiness` 同步增加
- 日任务次数和 `todayHappiness` 在跨日时重置
- 勋章进度不随跨日重置
- 最高等级时不再展示下一等级距离

## 洋流战报聚合规则

战报不维护首页的实时会话逻辑，只消费聚合结果与手动修正结果。

规则如下：

- 历史修正值必须满足合法时分秒范围
- 趋势图、统计卡和近 7 日明细必须共享同一份历史口径
- 切换 `income/duration` 或 `day/week/month/year` 只影响展示，不修改历史事实
- 单条记录允许被修正为 `00:00:00`
- 无数据区间必须给出空状态，而不是误导性百分比或涨跌提示

## 数据治理规则

数据管理中心是统一的数据治理入口。

规则如下：

- 导出成功或失败都不能改变原始业务数据
- 导出形式固定为 JSON 文本复制到剪切板，不生成下载文件
- 重置必须经过二次确认
- 重置若部分 key 删除失败，必须反馈失败，不得误报成功
- 本版不引入真实云端同步流程，云端同步卡片仅作占位展示

## 静态内容与外部入口规则

- 加入社区页必须指向官方入口，不得跳转未审核第三方地址
- 企业微信加群入口由 `materialPlugin` 和 `groupUrl` 承接
- 关于软件页至少展示当前版本号、更新日志和设计理念
- 更新日志首条版本号必须与头图区当前版本一致

## 跨页面责任边界

| 主题 | 事实源 | 主要写入页 | 主要消费页 |
| --- | --- | --- | --- |
| 航行档案 | `profile.settings` | 航行档案设置 | 首页、日历、战报、个人基地 |
| 头像 | `profile.avatar` | 个人基地 | 个人基地 |
| 今日会话 | `home.daily.session` | 首页 | 首页、战报 |
| 收益显隐 | `home.amount.visibility` | 首页 | 首页 |
| 岁月坐标 | `time.axis.entries` | 岁月坐标设置 | 首页、日历 |
| 快乐值与任务 | `lab.progress` | 动力室 | 动力室、个人基地、首页预览 |
| 战报修正 | `report.history.adjustments` | 洋流战报 | 洋流战报 |

## 占位能力清单

- 云端同步中心：仅展示“敬请期待”
- 社区分享反馈：允许占位，不强制实现复杂分享回传
- 战报高级筛选：本版不纳入
- 多端同步和账号体系：本版不纳入
