# 术语表

| 术语 | 定义 |
| --- | --- |
| 航行档案 | 用户的昵称、生日、入行日期、薪资、发薪日、工作制和作息配置，是全局基础输入。 |
| 今日会话 | 当日工作时长、避风时长、当前模式以及最后更新时间的合称。 |
| 收益快照 | 基于航行档案和今日会话派生出的时薪、秒薪、今日收益和月度累计摘要。 |
| 排班计划 | 工作制、大小周当前周、上下班时间和午休设置。 |
| 月游标 | 日历当前正在展示的年月。 |
| 日历状态 | 某日的工作日、休息日、节假日、补班日、发薪日等可视标记。 |
| 岁月坐标 | 一组重要日期记录的总称，用于维护纪念日、里程碑和倒计时。 |
| 坐标条目 | 岁月坐标中的单条记录，至少包含标题、日期、分组、图标和提醒模式。 |
| 周年提醒 | 同一条日期记录每年重复提醒的模式。 |
| 普通倒计时 | 面向单个固定日期的倒计时模式，允许显示已过天数。 |
| 动力室任务 | 用于累计快乐值的日常行为项，可设置次数上限与奖励。 |
| 快乐值 | 动力室的核心成长积分，用于推导等级与成长摘要。 |
| 等级摘要 | 基于快乐值推导出的当前等级、下一等级距离和进度。 |
| 洋流战报 | 收益与时长的趋势复盘页面。 |
| 近 7 日明细 | 战报中最近若干天的时长与收益记录，可被手动修正。 |
| 数据导出 | 将本地业务数据打包为 JSON 文本并复制到剪切板。 |
| 数据重置 | 清空全部业务存储的危险操作，必须经过二次确认。 |
| 云端同步中心 | 数据管理中心中的未来能力占位，本版仅展示，不触发真实流程。 |
| 官方社群 | 产品官方维护的社群入口，由加入社区页承接转化。 |
| 更新日志 | 关于软件页中按版本展示的变更记录。 |
