# 术语表

- `航行档案设置`：用户基础资料与排班设置页，对应 `ProfileSettings`
- `避风时长`：用户在工作日内记录的摸鱼时长，对应 `DailyVoyageRecord.moyuDurationSec`
- `航行日记录`：以自然日为粒度保存的核心事实记录，对应 `DailyVoyageRecord`
- `岁月坐标`：用户维护的重要事件记录，对应 `TimeAxisEntry`
- `档案本`：时间轴事件的系统分类，对应 `TimeAxisNotebook`
- `动力舱`：积分、任务、等级与成就系统，对应 `LabProgress`
- `启动门禁`：应用决定是否允许进入正常首页流程的初始化判断，对应 `AppBootstrapState`
- `初始化`：首次保存设置后创建默认业务数据和启动状态的过程
- `全量重置`：清除所有本地业务数据并恢复到未初始化状态的过程
- `派生视图`：基于核心实体计算出的页面展示结果，例如报表卡片、首页倒计时、日历提醒
