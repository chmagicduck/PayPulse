# 数据管理中心 Data Center

## 1. 页面目标

来源：页面直接事实 + 跨页推理

- 展示本地存储概览
- 提供数据导出与全量重置能力

## 2. 入口与前置条件

来源：页面直接事实 + 跨页推理

- 入口：个人基地 -> 数据管理中心
- 前置条件：已完成初始化
- 依赖：全部核心持久化数据

## 3. 信息结构

来源：页面直接事实

- 顶部导航
- 存储概览卡
- 云同步说明卡
- 导出动作项
- 重置动作项
- 风险提示卡
- 重置确认弹层

## 4. 核心交互

来源：页面直接事实 + 跨页推理

- 点击导出生成完整 JSON 备份
- 点击重置打开确认弹层
- 确认重置后清除全部本地业务数据和启动状态，并回到设置页

## 5. 页面状态

来源：页面直接事实

- `storageUsed`
- `storageCapacity`
- `storageUsedText`
- `storagePercentText`
- `exportStatus`
- `showResetModal`
- `resetModalVisible`
- `pressStates`

## 6. 依赖的共享数据 / 共享规则

来源：跨页推理

- 导出包结构受 `DataExportPackage` 约束
- 重置必须覆盖 `app.bootstrap.state`
- 页面上的容量和占比是所有 key 的汇总结果

## 7. 空态 / 异常态 / 边界情况

来源：闭环假设

- 导出失败时保留错误提示和重试入口
- 重置后不能停留在个人页或首页
- 存储为空时仍展示 0 值和正常导出按钮

## 8. 验收清单

- 导出内容完整覆盖核心实体
- 重置后所有业务数据消失并回设置页
- 重置不可撤销文案与真实行为一致
