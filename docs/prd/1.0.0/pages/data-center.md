# 数据管理中心

| 项 | 内容 |
| --- | --- |
| route | `features/data-center/pages/data-center` |
| prototype | `docs/prototype/1.0.0/features.data-center.pages.data-center.ts` |
| 主要入口 | 个人基地 |

## 页面目标

提供数据查看、导出和清理入口，帮助用户理解本地数据占用并安全执行敏感操作。

## 入口与前置条件

| 来源 | 条件 |
| --- | --- |
| 个人基地“数据管理中心” | 无 |

## 信息结构

| 顺序 | 区块 | 说明 |
| --- | --- | --- |
| 1 | 顶部导航 | 返回与标题。 |
| 2 | 存储指标 | 本地空间、占比和云端同步占位。 |
| 3 | 核心操作 | 导出备份、重置所有数据。 |
| 4 | 风险提示 | 说明本地优先与不可撤销性。 |
| 5 | 重置确认弹层 | 二次确认危险操作。 |

## 核心交互

| 动作 | 结果 |
| --- | --- |
| 点击导出备份 | 进入导出流程，成功时复制 JSON 到剪切板。 |
| 导出失败 | 反馈失败态，不影响现有业务数据。 |
| 点击重置所有数据 | 打开确认弹层。 |
| 取消重置 | 关闭确认弹层。 |
| 确认重置 | 清空正式业务 key，并反馈最终结果。 |

## 页面状态

| 状态 | 类型 | 说明 |
| --- | --- | --- |
| `exportStatus` | enum | `idle/exporting/success/failure`。 |
| `showResetModal` | boolean | 控制危险操作确认弹层。 |

## 依赖的共享数据 / 共享规则

- 读取微信存储 API 派生 `StorageUsage`。
- 导出与重置覆盖 1.0.0 正式业务 key。
- 云端同步中心只展示占位，不触发真实网络流程。
- 数据治理边界见 [domain-model.md](../domain-model.md) 与 [data-model.md](../data-model.md)。

## 空态 / 异常态 / 边界情况

- 无可导出数据时仍允许导出，但备份内容应为空对象或空集合。
- 导出形式固定为复制 JSON 文本，不生成下载文件。
- 重置过程中若部分 key 删除失败，必须返回失败提示，不得误报成功。
- 云端同步入口在本轮仅展示占位文案，不触发真实网络流程。

## 验收清单

- [ ] 存储占用数据可读且数值自洽。
- [ ] 导出有完整状态反馈。
- [ ] 重置必须经过确认，且执行后能清空业务数据。
- [ ] 占位中的云端同步不误导为可用功能。
