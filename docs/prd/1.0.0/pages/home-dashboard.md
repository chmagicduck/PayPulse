# 首页

| 项 | 内容 |
| --- | --- |
| route | `features/home/pages/dashboard` |
| prototype | `docs/prototype/1.0.0/features.home.pages.dashboard.ts` |
| 主要入口 | 默认 tab、冷启动 |

## 页面目标

让用户在一个页面内完成当日收益查看、计时操作、近期节点预览和人生进度感知。

## 入口与前置条件

| 来源 | 条件 |
| --- | --- |
| 底部导航默认 tab | 无 |
| 小程序冷启动 | 作为默认首页进入 |

## 信息结构

| 顺序 | 区块 | 说明 |
| --- | --- | --- |
| 1 | 收益总览 | 展示今日累计、时薪换算、月度累计与显示隐藏开关。 |
| 2 | 今日计时控制 | 展示下班倒计时、当前模式、工作/避风时长和修正入口。 |
| 3 | 潮汐任务预览 | 展示今日任务完成数与进度。 |
| 4 | 常规波动 | 展示发薪倒计时与休息日倒计时。 |
| 5 | 岁月坐标预览 | 展示重要日期并进入岁月坐标页。 |
| 6 | 人生航程看板 | 展示工龄、退休倒计时、总航程与余辉等摘要。 |
| 7 | 修正时长弹层 | 编辑避风时长。 |

## 核心交互

| 动作 | 结果 |
| --- | --- |
| 点击收益显隐 | 切换 `home.amount.visibility`，数字显示在明文和掩码间切换。 |
| 点击开始或停止避风 | 切换今日会话模式，并继续按秒推进当前累计时长。 |
| 计时每秒推进 | 更新 `home.daily.session`，同时刷新收益与倒计时展示。 |
| 点击修正入口 | 打开修正弹层。 |
| 保存修正时长 | 写回 `home.daily.session`，刷新汇总展示。 |
| 点击“管理全部” | 跳转岁月坐标设置页。 |
| 点击新增坐标入口 | 跳转岁月坐标设置页新建态。 |

## 页面状态

| 状态 | 类型 | 说明 |
| --- | --- | --- |
| `showEditModal` | boolean | 控制修正弹层显隐。 |
| `editDurationDraft` | object | 修正弹层的临时草稿值。 |

## 依赖的共享数据 / 共享规则

- 读取 `profile.settings` 推导收益、倒计时和人生摘要。
- 读取 `lab.progress` 展示任务完成摘要。
- 读取 `time.axis.entries` 展示近期重要日期。
- 读写 `home.daily.session` 维护今日工作与避风时长。
- 读写 `home.amount.visibility` 控制收益显隐。
- 收益口径、跨日重置、倒计时规则见 [domain-model.md](../domain-model.md)。

## 空态 / 异常态 / 边界情况

- 首次使用且缺少档案配置时，页面使用默认薪资与作息占位，并提示尽快补全档案。
- 当前时间晚于下班时间时，倒计时展示完成态，不显示负数。
- 岁月坐标为空时，展示空状态入口，不出现空白卡片。
- 当日会话跨日时，需要在进入页面或零点后重置到新的一天。

## 验收清单

- [ ] 收益、倒计时和计时数据能够自洽联动。
- [ ] 收益显隐与时长修正都能落盘。
- [ ] 空状态、跨日状态和下班后状态已处理。
- [ ] 岁月坐标与动力室数据仅做预览，不在首页内复制主数据。
