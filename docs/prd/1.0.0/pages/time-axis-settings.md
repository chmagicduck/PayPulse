# 岁月坐标设置

| 项 | 内容 |
| --- | --- |
| route | `features/time-axis/pages/time-axis-settings` |
| prototype | `docs/prototype/1.0.0/features.time-axis.pages.time-axis-settings.ts` |
| 主要入口 | 个人基地、首页“管理全部/新增坐标” |

## 页面目标

集中管理重要日期记录，使首页、日历等页面能够消费统一、可维护的时间节点数据。

## 入口与前置条件

| 来源 | 条件 |
| --- | --- |
| 个人基地“岁月坐标设置” | 无 |
| 首页“管理全部/新增坐标” | 无 |

## 信息结构

| 顺序 | 区块 | 说明 |
| --- | --- | --- |
| 1 | 顶部导航 | 返回、标题与新增入口。 |
| 2 | 分组筛选条 | 切换全部或某个分类。 |
| 3 | 坐标列表 | 展示图标、标题、日期、剩余天数和分组。 |
| 4 | 空状态 | 当前筛选下无条目时展示。 |
| 5 | 编辑抽屉 | 新增或编辑条目。 |

## 核心交互

| 动作 | 结果 |
| --- | --- |
| 点击分组 | 更新筛选结果。 |
| 点击新增 | 打开抽屉并重置草稿。 |
| 点击列表项 | 载入该条目并进入编辑态。 |
| 修改草稿字段 | 实时更新表单回显。 |
| 保存条目 | 写入 `time.axis.entries` 并刷新列表。 |
| 删除条目 | 从 `time.axis.entries` 删除目标项。 |
| 关闭抽屉 | 清空草稿并返回列表。 |

## 页面状态

| 状态 | 类型 | 说明 |
| --- | --- | --- |
| `selectedGroup` | string | 当前分组筛选条件。 |
| `isModalOpen` | boolean | 控制编辑抽屉。 |
| `editingId` | string \| null | 当前编辑条目 ID。 |
| `formDraft` | object | 编辑抽屉草稿。 |

## 依赖的共享数据 / 共享规则

- 读取并写入 `time.axis.entries`。
- 首页和日历只消费岁月坐标输出，不复制条目存储。
- 周年提醒、普通倒计时和公历限制见 [domain-model.md](../domain-model.md)。

## 空态 / 异常态 / 边界情况

- 当前筛选分组下无数据时，展示空状态和新增入口。
- 标题或日期为空时禁止保存。
- `isAnniversary=false` 且日期已过时，列表需明确显示“已过 N 天”而非负数。
- 本版移除农历选项，所有日期都按公历保存与展示。

- [ ] 新增、编辑、删除流程完整。
- [ ] 分组筛选与列表结果一致。
- [ ] 周年提醒和普通倒计时文案口径正确。
- [ ] 数据保存后能被首页和日历读取。
