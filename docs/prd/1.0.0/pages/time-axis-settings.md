# 时间轴设置页 Time Axis Settings

## 1. 页面目标

来源：页面直接事实 + 跨页推理

- 管理系统分类下的重要事件
- 为首页重要日期和日历提醒提供统一事实源

## 2. 入口与前置条件

来源：页面直接事实 + 跨页推理

- 入口：个人基地 -> 岁月坐标设置
- 前置条件：已完成初始化
- 依赖：`TimeAxisEntry`、系统 `TimeAxisNotebook`

## 3. 信息结构

来源：页面直接事实

- 顶部导航
- 档案本筛选条
- 事件列表或空态
- 新增档案入口卡
- 新增 / 编辑弹层

## 4. 核心交互

来源：页面直接事实 + 跨页推理

- 点击档案本切换筛选
- 点击条目进入编辑态
- 点击新增卡进入新增态
- 在弹层中编辑标题、日期、周年模式和分类
- 保存写入 `TimeAxisEntry`
- 删除移除对应事件

## 5. 页面状态

来源：页面直接事实

- `selectedNotebookId`
- `entries`
- `visibleEntries`
- `showEditModal`
- `editModalVisible`
- `editingId`
- `form`
- `pressStates`

## 6. 依赖的共享数据 / 共享规则

来源：跨页推理

- notebooks 为系统预置分类，页面只读
- 列表倒计时文案来自时间轴派生规则
- 首页重要日期与日历提醒共用这些事件

## 7. 空态 / 异常态 / 边界情况

来源：闭环假设

- 过滤后无数据时显示空态和新增入口
- 保存时标题或日期缺失则不允许提交
- 周年模式切换会改变同一条事件的倒计时口径

## 8. 验收清单

- 新增、编辑、删除后列表立即更新
- 切换分类只影响展示，不改底层数据
- 首页和日历相关派生展示随事件变化而变化
