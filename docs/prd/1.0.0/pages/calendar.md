# 日历

| 项 | 内容 |
| --- | --- |
| route | `features/calendar/pages/calendar` |
| prototype | `docs/prototype/1.0.0/features.calendar.pages.calendar.ts` |
| 主要入口 | 个人基地、首页或岁月坐标日期入口 |

## 页面目标

用月视图帮助用户查看发薪日、休息日、补班与节假日，并通过日期详情强化当前月节奏感知。

## 入口与前置条件

| 来源 | 条件 |
| --- | --- |
| 个人基地“查看日历” | 无 |
| 岁月坐标或首页日期入口 | 可带选中日期进入 |

## 信息结构

| 顺序 | 区块 | 说明 |
| --- | --- | --- |
| 1 | 顶部导航 | 返回与标题。 |
| 2 | 月份切换区 | 展示当前年月并支持翻页。 |
| 3 | 日历卡片 | 周标题、日期网格和图例。 |
| 4 | 选中日期详情 | 展示当前选中日的状态说明。 |
| 5 | 下一关键日提醒 | 展示距离下一发薪日或关键节点的提示。 |

## 核心交互

| 动作 | 结果 |
| --- | --- |
| 点击上一个月或下一个月 | 更新月游标并刷新网格。 |
| 点击具体日期 | 更新选中日期并刷新详情。 |
| 带日期入参进入 | 直接定位到目标日期所在月份。 |

## 页面状态

| 状态 | 类型 | 说明 |
| --- | --- | --- |
| `monthCursor` | string | 当前展示年月，建议格式 `YYYY-MM`。 |
| `selectedDate` | string | 当前详情区对应日期。 |

## 依赖的共享数据 / 共享规则

- 读取 `profile.settings` 推导工作制与发薪日。
- 读取 `time.axis.entries` 展示关键日期提示。
- 读取 2026 年法定节假日与补班预置常量。
- 日历状态口径、发薪日越界处理和大小周规则见 [domain-model.md](../domain-model.md)。

## 空态 / 异常态 / 边界情况

- 当前月没有节假日或补班配置时，仅展示普通工作日和发薪日逻辑。
- 当 `payDay` 大于当月天数时，发薪日取当月最后一天。
- 若用户尚未完成档案设置，则使用默认工作制与发薪日生成基础日历。
- 节假日与补班预置必须放在共享常量中，不得散落在页面模板。
