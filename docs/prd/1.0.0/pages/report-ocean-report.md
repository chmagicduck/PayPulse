# 洋流战报

| 项 | 内容 |
| --- | --- |
| route | `features/report/pages/ocean-report` |
| prototype | `docs/prototype/1.0.0/features.report.pages.ocean-report.ts` |
| 主要入口 | 首页、底部导航、个人基地 |

## 页面目标

为用户提供收益与时长的复盘视角，支持查看趋势、占比、阶段成就，并修正近期历史记录。

## 入口与前置条件

| 来源 | 条件 |
| --- | --- |
| 首页或底部导航“洋流战报” | 无 |
| 个人基地快捷入口 | 无 |

## 信息结构

| 顺序 | 区块 | 说明 |
| --- | --- | --- |
| 1 | 页面头部 | 标题与筛选入口占位。 |
| 2 | 趋势图 | 收益/时长双维度切换的主图表。 |
| 3 | 年度成就 | 展示年度累计收益、天数、排名等摘要。 |
| 4 | 时长占比统计 | 展示不同时间范围的工作/避风占比。 |
| 5 | 近 7 日明细 | 展示日级时长和收益，可进入修正。 |
| 6 | 修正弹层 | 编辑单条历史时长。 |

## 核心交互

| 动作 | 结果 |
| --- | --- |
| 切换收益或时长维度 | 切换趋势图和总计口径。 |
| 切换统计范围 | 切换当前聚合区间。 |
| 点击历史明细 | 打开修正弹层并载入该条记录。 |
| 保存修正时长 | 写入 `report.history.adjustments` 并刷新列表与统计。 |
| 关闭修正弹层 | 清空草稿并返回列表。 |

## 页面状态

| 状态 | 类型 | 说明 |
| --- | --- | --- |
| `activeMetric` | enum | `income/duration` 双维度切换。 |
| `timeRange` | enum | `day/week/month/year` 统计范围。 |
| `editingIndex` | number \| null | 当前编辑的明细索引。 |
| `editDurationDraft` | object | 修正弹层草稿值。 |
| `showEditModal` | boolean | 控制修正弹层显隐。 |

## 依赖的共享数据 / 共享规则

- 读取 `profile.settings` 换算收益。
- 读取 `home.daily.session` 和 `report.history.adjustments` 生成趋势、统计与近 7 日明细。
- 写入 `report.history.adjustments` 保存历史修正。
- 历史口径一致性、零值处理和展示边界见 [domain-model.md](../domain-model.md)。

## 空态 / 异常态 / 边界情况

- 无历史记录时，趋势图、成就区和明细区都要展示空状态说明。
- 单条记录被修正为 `00:00:00` 时允许保存，但收益应为 `0` 或按统一规则重算。
- 统计范围切换到无数据区间时，不显示误导性的百分比或涨跌提示。

## 验收清单

- [ ] 趋势维度和统计范围切换都能正确刷新展示。
- [ ] 历史修正后，列表与汇总值保持一致。
- [ ] 空状态和零值状态可读。
- [ ] 页面没有重复维护首页的实时会话状态，只消费聚合结果。
