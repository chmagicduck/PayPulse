# 验收清单

## 功能

- [ ] 可切换月份并正确渲染月视图。
- [ ] 可选择具体日期并查看详情说明。
- [ ] 发薪日、休息日、补班日和节假日能用清晰状态区分。
- [ ] 下一关键日提示与日历状态口径一致。
- [ ] 2026 年法定节假日与调休日展示正确。

## UI

- [ ] 日历网格、图例和详情卡层级清晰。
- [ ] 选中态、发薪态和节假日态有稳定的主题语义色。
- [ ] 无特殊日期的月份仍保持可读，不因数据稀疏而失真。

## 可维护性

- [ ] 日历状态来自档案配置和派生规则，不在页面里硬编码全量业务逻辑。
- [ ] 图例文案与状态码一一对应。
- [ ] 页面只维护月游标和选中日期等局部状态。
