# PayPulse

仓库协作说明仅维护在 `AGENTS.md`。

## 项目分层架构

```text
|------------------------------------------------------------------|
| 应用入口层 App Entry                                              |
|------------------------------------------------------------------|
| miniprogram/app.ts                                                |
| miniprogram/app.json                                              |
| miniprogram/app.less                                              |
|                                                                  |
| 负责小程序注册、全局页面声明、全局样式与运行时入口。               |
|------------------------------------------------------------------|

                               |
                               v

|------------------------------------------------------------------|
| 展示层 Presentation                                               |
|------------------------------------------------------------------|
| miniprogram/features/*/*.wxml                                     |
| miniprogram/features/*/*.less                                     |
| miniprogram/components/ui/*                                       |
|                                                                  |
| 负责页面结构、视觉样式、共享 UI 组件。                            |
|------------------------------------------------------------------|

                               |
                               v

|------------------------------------------------------------------|
| 页面编排层 Page Controller                                        |
|------------------------------------------------------------------|
| miniprogram/features/*/*.ts                                       |
|                                                                  |
| 负责生命周期、交互事件、导航、setData 串联。                      |
| 不在这里堆复杂业务规则与大块静态文案。                            |
|------------------------------------------------------------------|

                               |
                               v

|------------------------------------------------------------------|
| Feature 视图模型层 ViewModel / Helper                             |
|------------------------------------------------------------------|
| miniprogram/features/*/helper/*                                   |
| miniprogram/features/*/model/index.ts                             |
| miniprogram/features/*/model/view.ts                              |
| miniprogram/features/*/model/state.ts                             |
| miniprogram/features/*/model/types.ts                             |
|                                                                  |
| 负责页面展示数据拼装、静态视图骨架、局部类型和运行时状态输出。     |
|------------------------------------------------------------------|

                               |
                               v

|------------------------------------------------------------------|
| Feature 动作与持久化层 Actions / Storage                          |
|------------------------------------------------------------------|
| miniprogram/features/*/model/actions.ts                           |
| miniprogram/features/*/model/storage.ts                           |
|                                                                  |
| 负责页面级副作用、增删改、草稿处理和本地缓存读写。                |
|------------------------------------------------------------------|

                               |
                               v

|------------------------------------------------------------------|
| 领域层 Domain                                                     |
|------------------------------------------------------------------|
| miniprogram/lib/domain/date.ts                                    |
| miniprogram/lib/domain/daily-records.ts                           |
| miniprogram/lib/domain/calendar.ts                                |
| miniprogram/lib/domain/retirement.ts                              |
| miniprogram/lib/domain/moyu-session.ts                            |
| miniprogram/lib/domain/lab-progress.ts                            |
| miniprogram/lib/domain/lab-tasks.ts                               |
|                                                                  |
| 负责工资、日历、退休、摸鱼会话、任务进度等核心业务规则。           |
|------------------------------------------------------------------|

                               |
                               v

|------------------------------------------------------------------|
| 基础设施层 Infrastructure                                         |
|------------------------------------------------------------------|
| miniprogram/lib/wx/*                                              |
| miniprogram/lib/constants/*                                       |
| miniprogram/store/bootstrap.ts                                    |
| miniprogram/store/storage-registry.ts                             |
| miniprogram/store/app-store.ts                                    |
|                                                                  |
| 负责微信能力封装、初始化、导出、重置、存储治理与应用级状态。       |
|------------------------------------------------------------------|

                               |
                               v

|------------------------------------------------------------------|
| 配置与设计资源层 Config / Theme / Assets                          |
|------------------------------------------------------------------|
| miniprogram/config/*                                              |
| miniprogram/theme/*                                               |
| miniprogram/assets/*                                              |
|                                                                  |
| 负责路由事实源、tab 配置、设计 token、图标与图片资源。            |
|------------------------------------------------------------------|
```

更细的目录职责、文件边界和协作提示见 `AGENTS.md`。
